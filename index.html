RiftPDB


Project Description:

3DProteinViewer (now RiftPDB) will display the 3D ribbon model of the inputted protein (using a PDB code). The specified protein structure will be downloaded from the Protein Data Bank which will then be interpreted and displayed using the ChemDoodle Web Components API (WebGL/Three.js). The ultimate goal of this project is to make all of this operable within the web browser on the Oculus Rift allowing the user to view the model in virtual reality.

Example of Oculus Rift function in a web browser

Extended Goals:

  • Allow manipulation of the protein using keyboard/mouse
  • Display multiple proteins at once
